									<h3><span style="color:#003a70;"><span style="font-size:18px;">An Update on Memory Safety in the Linux Kernel</span></span></h3>

									<p><a href="https://outreach.abetterinternet.org/e/1011011/2025-03-06/3614y/%%email_id%%/h/%%internal.tracker_url_hash%%">Rust for Linux</a> is an effort to support the use of the Rust language in the Linux kernel. Memory safety errors account for a large portion of kernel vulnerabilities, but this can be reduced as more drivers are written in a memory safe language. In 2021, we started working with the project's primary maintainer, <a href="https://outreach.abetterinternet.org/e/1011011/2025-03-06/36152i/%%email_id%%/h/%%internal.tracker_url_hash%%">Miguel Ojeda</a>, shortly after he published the <a href="https://outreach.abetterinternet.org/e/1011011/4604-23473-1-ojeda-kernel-org-/36155i/%%email_id%%/h/%%internal.tracker_url_hash%%">original RFC</a> for Rust in the Linux kernel<br>
